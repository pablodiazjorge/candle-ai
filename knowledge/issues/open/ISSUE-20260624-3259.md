## Description

In Angular standalone components, every pipe used in the template must be explicitly added to the component's `imports` array, even if the pipe class is provided by `@angular/common`. Unlike NgModule-based components, standalone components have no implicit access to common pipes.

Error: `NG8004: No pipe found with name 'date'.`

Commonly forgotten: `DatePipe`, `DecimalPipe`, `PercentPipe`, `SlicePipe`.

## Evidence

- Session 68e8a9c8 (Phase 4): In `analysis-dashboard.html`, `{{ a.generatedAt | date:'medium' }}` failed because `DatePipe` was not in the component's `imports` array. `CurrencyPipe` was already imported but `DatePipe` was forgotten.
