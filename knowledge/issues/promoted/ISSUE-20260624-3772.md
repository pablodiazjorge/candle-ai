## Description

When `isolatedModules` is enabled in `tsconfig.json` (Angular default), re-exporting a TypeScript type inline with regular exports fails with `TS1205`. Must use `export type { Foo }` separately from `export { bar }`.

Example error:
```
TS1205: Re-exporting a type when 'isolatedModules' is enabled requires using 'export type'.
```

## Evidence

- Session 68e8a9c8: In `indicators.service.ts`, `export { IndicatorSettings, DEFAULT_INDICATOR_SETTINGS }` failed because `IndicatorSettings` is a type. Fixed by splitting: `export type { IndicatorSettings }; export { DEFAULT_INDICATOR_SETTINGS };`
